Emoji Mirror
An interactive, AI-powered web application that uses computer vision to map user facial expressions onto 3D rendered emojis in real-time. This project represents the bridge between my foundational logic in Java and my transition into Full-Stack AI development.

🛠️ Technical Architecture
The Engine
React 19 & Vite: Utilized for a high-performance, reactive UI and lightning-fast build cycles.

Three.js: Handles the 3D scene, rendering the "Mirror" objects with realistic lighting and materials.

MediaPipe Vision: Implements deep learning models for real-time facial landmark detection.

Core Logic
The project applies structured problem-solving habits developed through my Core Java studies:

Encapsulation: Keeping 3D scene logic separate from the AI tracking data.

Modular Methods: Clean function structures for processing raw coordinates into emoji transformations.
